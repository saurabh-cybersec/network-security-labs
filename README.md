Network Security Labs

This repository contains hands-on networking and low-level security labs built using Python and Linux.

The goal of these labs is to understand how protocols and packet communication work at different layers of the network stack through practical implementation and packet analysis.

Current labs include:

TCP client-server communication
UDP client-server communication
Raw Ethernet frame crafting
ARP packet crafting
MAC spoofing
MAC flooding
MITM experimentation

Tools and technologies used:

Python sockets
Raw sockets (AF_PACKET)
Wireshark
Linux virtual machines
VMware networking
